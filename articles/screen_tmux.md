# Screen

`screen` — это один из самых полезных инструментов для работы в терминале Linux. Он позволяет создавать несколько виртуальных терминалов в одном окне, а главное — процессы продолжают работать даже после закрытия SSH-соединения.

## Зачем нужен screen?

Представьте ситуацию: вы запустили долгий процесс на сервере через SSH, а через час интернет отключился. Процесс останавливается, и все приходится начинать заново. 😱

`screen` решает эту проблему:

- ✅ Процессы не завершаются при разрыве соединения
- ✅ Можно работать в нескольких терминалах одновременно
- ✅ Легко переключаться между сессиями
- ✅ Можно делиться терминалом с другими пользователями

Все управляющие команды в **screen** начинаются с Ctrl+a:

|Команда|Назначение|
|:-:|:-:|
|(Ctrl+a) + c|Создать новое окно|
|(Ctrl+a) + w|Посмотреть список созданных окон|
|(Ctrl+a) + номер_окна или "|Выбор окна|
|(Ctrl+a) + n или p|Переход последовательно между окнами|
|(Ctrl+a) + S|Разделить окно по горизонтали|
|(Ctrl+a) + \||Разделить окно по вертикали|
|(Ctrl+a) + Tab|Для переключения между сплит-окнами|
|(Ctrl+a) + X|Скрыть текущий экран|
|(Ctrl+a) + Q|Скрыть все остальные экраны|
|(Ctrl+a) + A|Переименовать экран|
|(Ctrl+a) + d|Выход из менеджера окон|
|(Ctrl+a) + ?|Список команд|

``` bash
screen -ls # Посмотреть список запущенных сессий
screen -S name # Создать окно с именем
screen -r id или name # Подключиться к сессии
```
