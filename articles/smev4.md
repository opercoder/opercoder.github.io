# (В процессе наполнения) СМЭВ4

## Процесс подключения к СМЭВ4. Опыт развертывания и встречающиеся ошибки  

На момент создания статьи существует 3 стенда: для разработчиков, тестовый и продуктивный.  
Для подключения к продуктивному контуру потребуется подключения по защищенному каналу связи и сертификаты, выданные УЦ, который загружаются в ЛК УВ. Для подключения к первым двум - сертификат и криптоконтейнер Крипто Про создаются онлайн с возможностью скачивания.  

## ЛК УВ

1. Регистрация информационной системы (далее - ИС).  
1.1 Быстрые действия - Информационные системы - Добавить систему - ВЫБРАТЬ СРЕДУ - новая - для прода загрузить сертификат / для тест сгенерировать - скачать архив с контейнером, запомнить имя и пароль - потом можно будет найти систему. 
1.2 На основной вкладке системы нажать редактировать и ВЫБРАТЬ РОЛЬ (потребитель или поставщик) - можно скачать пакет установочный.
